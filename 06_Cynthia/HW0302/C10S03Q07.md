# Solution
Given the following sample data:

**Population 1:** $\{12, 3, 8, 5\}$  
**Population 2:** $\{14, 7, 7, 9, 6\}$

## Summary Statistics Calculation:
* **Population 1:**
  * Sample Size ($n_1$): 4
  * Sample Mean ($\bar{x}_1$): $(12+3+8+5)/4 = 7.0$
  * Sum of Squares ($SS_1$): $(12-7)^2 + (3-7)^2 + (8-7)^2 + (5-7)^2 = 25 + 16 + 1 + 4 = 46$
  * Sample Variance ($s_1^2$): $SS_1 / (n_1 - 1) = 46/3 \approx 15.333$

* **Population 2:**
  * Sample Size ($n_2$): 5
  * Sample Mean ($\bar{x}_2$): $(14+7+7+9+6)/5 = 8.6$
  * Sum of Squares ($SS_2$): $(14-8.6)^2 + (7-8.6)^2 + (7-8.6)^2 + (9-8.6)^2 + (6-8.6)^2 \approx 29.16 + 2.56 + 2.56 + 0.16 + 6.76 = 41.2$
  * Sample Variance ($s_2^2$): $SS_2 / (n_2 - 1) = 41.2/4 = 10.3$

Null Hypothesis ($H_0$): $\mu_1 - \mu_2 = 0$
Alternative Hypothesis ($H_a$): $\mu_1 - \mu_2 \neq 0$

$$s_p^2 = \frac{(n_1 - 1)s_1^2 + (n_2 - 1)s_2^2}{n_1 + n_2 - 2}$$

$$s_p^2 = \frac{(3)(15.333) + (4)(10.3)}{4 + 5 - 2} = \frac{46 + 41.2}{7} = \frac{87.2}{7} \approx 12.4571$$

$$df = n_1 + n_2 - 2 = 4 + 5 - 2 = 7$$

$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{s_p^2 \left( \frac{1}{n_1} + \frac{1}{n_2} \right)}}$$

$$t = \frac{7.0 - 8.6}{\sqrt{12.4571 \left( \frac{1}{4} + \frac{1}{5} \right)}} = \frac{-1.6}{\sqrt{12.4571(0.45)}} \approx \frac{-1.6}{2.3676} \approx -0.6758$$

For a two-tailed test with $\alpha = 0.05$ and $df = 7$:
Critical Value ($t_{crit}$): $\pm 2.365$
Rejection Region: Reject $H_0$ if $|t| > 2.365$.
p-value: $\approx 0.5208$ (calculated via t-distribution)
